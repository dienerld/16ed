Css

Seletores

tag - id - class

Tag

Para estilizar todos as ocorrências de uma tag podemos passar apenas a tag(elemento) dentro do css e definir nossos estilos Por exemplo - span {color: green;}

Id

Para estilizar um item com id devemos usar o prefixo # antes do id - por exemplo este paragrafo tem o id p1. Para estiliza-lo devo usar #p1 para seleciona-lo para estilização

Class

Para estilizar um item com class devemos usar o prefixo >.< antes do nome da classe - por exemplo este paragrafo tem a classe paragrafo. Para estiliza-lo devo usar .paragrafo para seleciona-lo para estilização

Propriedades Css

  • height - Definir altura
  • width - Definir largura
  • color - Definir cor
  • background-color - Definir cor de fundo
