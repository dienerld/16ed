# Api Carros

Api para oficina mecânica.

## Requisitos Funcionais

1. Criar Cliente
2. Criar Carros
3. Gerar ordem de serviço
4. Atualizar ordem de serviço
5. Remover carro
6. Atualizar carro
7. Atualizar cliente

### Carro

- Marca
- Ano
- Cor
- Modelo
- Placa
- Defeitos

### Ordem de serviço

- Cliente
- Orçamento
- Prazo
- Carro
- Status conclusão

### Cliente

- Nome
- CPF
- Endereço
- Contato
- Carros
